# Task: Build AgentForge v0.1 MVP

## Task Goal
Produce a working Next.js 14 repo that lets an AI Agent consultant run their sales / design / development / delivery loop end-to-end: capture leads, run AI-driven diagnostic reports, design workflow blueprints, generate deliverable bundles from a curated template library, and track project revenue / tickets.

## Required Files
- `docs/product-spec.md` — what to build and what to skip.
- `docs/architecture.md` — how the pieces connect.
- `src/lib/schema.ts` — single source of truth for tables and enums.
- `src/lib/templates.ts` — the value-bearing IP. Edit here, not in scattered files.

## Known-Fact Summary
- Stack: Next.js 14 App Router + TypeScript + Tailwind + better-sqlite3 + Anthropic SDK + archiver. No other npm deps.
- Single SQLite file under `data/`. No multi-user auth in v0.1.
- Diagnostic reports use Claude (`generateDiagnostic`) or a deterministic `fallbackDiagnostic` when no API key.
- Bundler outputs zip with python script + n8n JSON + README + requirements + DELIVERY.md.

## Forbidden Actions
- Do not add new top-level deps beyond the five above.
- Do not generate `*.md` planning docs unless explicitly asked.
- Do not break the share-token public route by re-introducing `<html>`/`<body>` in child layouts.

## Validation Commands
- `npm install`
- `npm run typecheck`
- `npm run build`
- `npm run seed`

## Checkpoint Format
Append one JSON line to `.agent/state/MVP-001.json` after each phase: `{ "phase": "...", "done": true, "at": "<iso>" }`. Emit `AGOS_TASK_DONE` as the last stdout line when the whole task is shipped.
